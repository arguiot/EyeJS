## Getting Started

To use EyeJS, you'll need to download EyeJS via NPM, and create a `__test__`
folder in your project folder, and put tests files in it. (NPM might require
`sudo` on Mac).

```bash
$ npm install -g eye.js
```

## Run EyeJS

EyeJS comes with a CLI:

```bash
$ eye
```
